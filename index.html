What Does The Data Represent?


The pie charts above are used to represent the relative amount of time users spend on each section of my website.
The user activity is first collected and used to update the user's chart in real-time. Afterwards, it is written to Firebase's real-time database at the end of each session. Each time the website is accessed, data is read from the database and used to display the Avg Time per Section % pie chart.

Mini Quadcopter




The Mini Quadcopter was one of the most technically challenging hardware projects I've done. Weight had to be minimized, so I chose to place the circuitry on protoboard using an MPU6050 serving as gyro and accelerometer and a Arduino Micro as its microcontroller. The drone is powered by a 3.7v lithium battery.

Ping Pong Turret




For my culminating task in grade 12 tech engineering class, I made a ping pong ball turret that was wirelessly controlled. The command is sent via ssh from a thrid party app on my mobile device, then the Raspberry Pi interprets and executes the command. The stepping motor controls the turrets rotations and the servo releases the ball from the collector.
